# External Merge Sort

## Description
This project demonstrates sorting of large files that do not fit into memory using External Merge Sort.

## How it works
1. Split the file into smaller chunks
2. Sort each chunk in memory
3. Merge all sorted chunks into a final sorted file

## Concepts Used
- Divide and Conquer
- File Handling
- Merge Sort

## Example
Large file → chunks → sorted chunks → merged output
